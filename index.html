HTML Cheat-sheet

Tag What does it do?
<head> Delimits the page's head
<title> Page title (does not show on the page)
<b> Sets text to bold
<i> Sets text to italics
<center> Center aligns, horizontally
<ul> Un-ordered list, bullets.
<ol> Ordered list
<li> Numbered list
<br> Line break
<p> Paragraph starter
<img src="..."> Displays an image
<href="..."> Defines a hyper link referance
<tr> Table row
<td> Table row information
